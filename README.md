# TIVIT - .Net com GitHub Copilot (Bootcamp)

Introdução ao .NET, história, versões e usos

## Introdução:

O .NET é uma plataforma de desenvolvimento unificado que permite a construção de sistemas e aplicações.

C# é uma linguagem de programação usada no .NET.

Por exemplo, eu quero construir uma aplicação WEB usando ASP NET, com isso o .NET vai me fornecer as coisas que eu preciso para construir aquela aplicação, e C# vai ser a maneira de como vou construir.

## História do .NET:

A Microsoft começou a trabalhar no C# no final dos anos 90, tendo sua primeira versão do framework lançada em 2002, tendo como objetivo competir com o Java.

O conceito de multiplataforma era estrondoso na época, e a Microsoft trabalhou em melhorias e implementações do Java em sua plataforma, mas foi impedida pela Sun.

Isso motivou a Microsoft a desenvolver um ecossistema integrado ao Windows, que fosse fácil para desenvolver aplicativos desktop e web, e que e fosse amarrado apenas ao Windows, aumentando assim a sua relevância.

## Diferenças entre .NET Framework (Legado) e .NET ou .NET Core

### .NET Framework
- Funciona apenas no Windows

### .NET Core 5
- Foi feito para ser multiplataformas
- Facilidade em desenvolver para múltiplos sistemas
- Baratear muito mais os custos, em comparação entre contratar um servidor Windows x Linux, com Linux o servidor fica muito mais barato.
- Posso tranquilamente desenvolver um aplicativo em Linux para rodar em Windows, ou vice-versa.
